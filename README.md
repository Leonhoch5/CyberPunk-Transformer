# **2D Fighting Game**

## **Overview**

A fast-paced 2D fighting game using HTML5 Canvas and JavaScript. Face off against an AI with responsive animations, health tracking, and scorekeeping.

## **Features**

- Dynamic animations and smooth gravity-based movement.
- Combat system with attacks, AI behavior, and collision detection.
- Main menu, pause menu, and match-winning screens.
- FPS counter and responsive design.

## **Controls**

- **Move**: A/D
- **Jump**: W
- **Crouch**: S
- **Attack**: F
- **Start/Continue**: Enter
- **Pause**: P

## **How to Play**

Defeat the AI by reducing its health to 0. First to 7 points wins the match.
